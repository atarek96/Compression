# Lossless Data Compression Techniques

A **Tkinter-based GUI** application that demonstrates and compares various **Lossless Data Compression** algorithms. Built as an educational tool to visualize how different compression techniques work.

## ✨ Features

- **Interactive GUI** built with Tkinter
- Support for **5 popular compression techniques**:
  - Run-Length Encoding (RLE)
  - Huffman Encoding
  - Arithmetic Encoding
  - Golomb Encoding
  - LZW Encoding
- Real-time calculation of key metrics:
  - Bits Before & After Encoding
  - Compression Ratio (%)
  - Entropy
  - Average Code Length
  - Compression Efficiency (%)

## 🛠 Technologies Used

- **Python 3**
- **Tkinter** (for GUI)
- **heapq** (for Huffman Coding)
- **collections.Counter**
- **math**
